- Paquetes Octave necesarios: optim, struct, io, statistics.
>> pkg install -forge struct
>> pkg install -forge io
>> pkg install -forge optim
>> pkg install -forge statistics
- Comprobar vectores de los modelos.
- Para octave hay que compilar con "make octave" desde terminal o con "make" desde octave.
- cell de más longitud en Utilities processDirectory

Para instalar octave y las librerías de desarrollo desde backports
$ sudo apt-get -t jessie-backports install octave
$ sudo apt-get -t jessie-backports install liboctave-dev 

Para instalar octave desde anaconda (recomendable hacerlo desde un environment de conda distinto):
$ conda install -c conda-forge octave
